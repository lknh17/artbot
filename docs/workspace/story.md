# 雨中的咖啡馆

窗外的雨淅淅沥沥地下着，林悦坐在街角咖啡馆的靠窗位置，手里捧着一杯已经凉透的美式咖啡。

今天是她和陈默分手的第三十天。三十天前，也是这样一个雨天，陈默在这个咖啡馆里对她说："我们都需要冷静一下。"

林悦苦笑了一下。所谓的冷静，其实就是分开的另一种说法。

她抬起头，透过被雨水模糊的玻璃窗，看见街对面站着一个熟悉的身影——是陈默。他撑着一把黑色的伞，也在看着这家咖啡馆。

两个人的目光隔着雨幕相遇了。

陈默犹豫了一下，然后穿过马路，推开了咖啡馆的门。风铃发出清脆的声响。

"你还好吗？"他问。

"还好。"林悦低下头，"你呢？"

"不太好。"陈默在她对面坐下，"这三十天我想明白了一件事。"

"什么？"

"冷静期不是分开，是为了让我们更好地在一起。"他从口袋里掏出一个小盒子，"林悦，我想重新追求你，从一杯热咖啡开始，可以吗？"

林悦看着那个盒子，眼眶湿润了。

窗外的雨还在下，但咖啡馆里，慢慢暖和了起来。
